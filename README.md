# Project Name: India's Iconic Cars

## Description
This project is a simple **HTML-only website** that showcases articles about **India’s iconic cars**. It highlights their history, features, and significance in Indian automotive culture. The project demonstrates the use of HTML elements like headings, paragraphs, lists, images, links, tables, and forms—all without CSS or JavaScript.

## Features
- Fully built with **HTML5**
- Informative articles about iconic Indian cars
- Headings and subheadings for clear content structure
- Ordered and unordered lists to organize car features
- Images of iconic cars embedded from local storage
- Table showcasing car specifications
- External links to car-related resources
- Contact form for user feedback (HTML-only, no backend)

## Example Cars Featured 
-Hindustan Ambassador
- Premier Padmini
- Hindustan Motors Contessa  
-Mahindra Jeep (CJ / MM540)
## Technology Used

- **HTML5** – The core language used to build the structure and content of the website.
